# ✦ AURÉA ✦

### *Elegance Is An Attitude*

Landing page untuk brand fashion mewah AURÉA — di mana setiap piksel dirancang dengan kepekaan yang sama seperti setiap jahitan pada koleksinya.

</div>

---

## Tentang Proyek

**AURÉA** bukan sekadar landing page — ini adalah interpretasi digital dari filosofi *timeless elegance*. Dibangun dari nol, mulai dari rancangan visual di Figma hingga implementasi kode, dengan satu tujuan: menghadirkan kemewahan yang terasa personal, bukan generik.

## Struktur Halaman

- **Hero** — pernyataan sikap, bukan sekadar sambutan
- **Our Story** — jejak perjalanan craftsmanship
- **Collection** — koleksi terkini, disusun dengan presisi
- **Testimonials** — suara para pelanggan
- **Footer** — penutup yang tetap berkelas

## Dibangun Dengan

`HTML5` · `CSS3` · `Vanilla JavaScript`

Tanpa framework berat — kesederhanaan yang disengaja, seperti prinsip *less is more* dalam desain busana.

## Menjalankan Secara Lokal

```bash
git clone https://github.com/ArmansyahSoft/Aurea.git
cd Aurea
```

Buka `index.html` di browser — atau akses versi langsungnya lewat GitHub Pages *(tautan menyusul)*.
